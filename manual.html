이 웹사이트를 어떻게 사용하면 되나요?

이 사이트는 `2026 초등부 4~6 믿음 캠프`를 소개하고, 캠프 전·중·후에 필요한 AI 도구와 입력 기능을 한곳에서 사용할 수 있도록 만든 안내형 웹사이트입니다. 먼저 상단 소개와 포스터·영상으로 캠프 분위기를 확인한 뒤, 필요한 도구나 입력 기능으로 이동하면 됩니다.

MAIN PAGE

메인 화면에서 할 수 있는 일

캠프 소개 보기 다니엘의 믿음이라는 캠프 주제와 대상, 핵심 메시지를 빠르게 이해할 수 있습니다.
포스터와 영상 보기 홍보 포스터와 소개 영상을 한눈에 보며 캠프 분위기를 먼저 파악할 수 있습니다.
사용할 교재 보기 DAY 1, DAY 2, DAY 3 교재를 겹쳐진 카드 미리보기로 보고 필요한 이미지를 따로 열 수 있습니다.
AI 도구 살펴보기 캠프 전, 캠프 중, 캠프 후에 맞는 도구들을 섹션별로 찾아 바로 열 수 있습니다.

AI TOOLS

도구 섹션은 이렇게 보면 됩니다

  • 캠프 전: 신청, 조 편성, 차량, 준비물, 홍보물, 찬양 등 준비 단계 도구입니다.
  • 캠프 중: 게임, 퀴즈, 일정 공유, 기도 지원 등 현장 진행용 도구입니다.
  • 캠프 후: 소감문, 사진 정리, 영수증, 기도제목 관리 등 마무리 도구입니다.
  • 각 카드의 `도구 열기` 버튼을 누르면 해당 기능이 새 탭에서 열립니다.

INPUT FORMS

입력 기능은 언제 사용하나요?

  • 캠프 신청 입력: 학생과 보호자 기본 정보를 남길 때 사용합니다.
  • 소감문 입력: 캠프 후 받은 은혜와 느낀 점을 기록할 때 사용합니다.
  • 기도제목 입력: 함께 기도받고 싶은 내용을 남길 때 사용합니다.
  • 입력 후 상태 문구가 보이며, 실제 저장은 연결된 Apps Script 설정에 따라 동작합니다.

BEST ORDER

추천 사용 순서

  • 1. 메인 상단에서 캠프 소개 문구를 읽습니다.
  • 2. 포스터, 소개 영상, 사용할 교재를 통해 캠프 자료를 먼저 확인합니다.
  • 3. 필요한 시점에 맞춰 `캠프 전`, `캠프 중`, `캠프 후` 도구를 선택합니다.
  • 4. 마지막으로 신청, 소감문, 기도제목 입력 기능을 사용합니다.
